# 📊 Superstore Sales Analysis

This project analyzes sales data from a global superstore to extract actionable business insights.

## 🔍 Dataset Info
- Rows: ~10,000
- Columns: 21
- Fields include: Order Date, Ship Date, Region, Category, Sub-Category, Sales, Profit, Discount, etc.

## 📈 Key Analyses
- Sales & profit by category and region
- Monthly trend analysis
- Customer segmentation by segment and region
- Top 10 profitable products

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
